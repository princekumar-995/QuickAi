# 🚀 Quick.ai — AI Powered Developer & Productivity Platform

Quick.ai is a futuristic AI-powered SaaS platform designed for developers, creators, and students to build smarter using AI automation tools. The platform combines AI workflow generation, AI blogging, AI chatbot assistance, resume intelligence, AI image generation, and background removal into one modern ecosystem with a premium futuristic UI/UX.

⚙️ Installation & Setup
1️⃣ Clone Repository
git clone https://github.com/your-username/QuickAI.git

2️⃣ Install Frontend Dependencies
cd client
npm install

3️⃣ Install Backend Dependencies
cd ../server
npm install

🔑 Environment Variables
Backend .env
PORT=3000

OPENAI_API_KEY=your_openai_key

OPENROUTER_API_KEY=your_openrouter_key

DATABASE_URL=your_database_url

CLERK_SECRET_KEY=your_clerk_secret_key

CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_cloud_api_key
CLOUDINARY_API_SECRET=your_cloud_api_secret

Frontend .env
VITE_API_BASE_URL=http://localhost:3000

VITE_CLERK_PUBLISHABLE_KEY=your_clerk_publishable_key

▶️ Run Project
Start Backend Server
cd server
npm run dev

Backend will run on:
http://localhost:3000
Frontend will run on:
http://localhost:5173
